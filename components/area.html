省市区选择 ​

l-ui 推荐使用 element-plus 作为基础组件。

  • 基于element-plus@2.5.3 @element-plus/icons-vue@2.3.1版本进行组件的封装。
sh
pnpm install element-plus @element-plus/icons-vue

使用组件 ​

  • 如果你想像用例一样直接使用，你需要全局注册组件，才能够直接在项目里使用。

API ​

Attributes ​

参数说明类型可选值默认值
model-value / v-model从选中项绑定值string[]|number[]--
options选项的数据源， value 和 labelobject--
clearable是否支持清空选项boolean-false
show-all-levels输入框中是否显示选中值的完整路径boolean-true

Event ​

参数说明类型参数
change当绑定值变化时触发的事件function(value)
