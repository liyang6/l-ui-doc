绘制点 ​

根据经纬度绘制点，添加点击事件，点击元素后并提示popover插槽内容信息

点名称 :
点ID :
经度 :
纬度 :
显示代码
vue
<script setup lang="ts">
import { ref, onMounted } from 'vue'
import type { Geometry, Point } from 'ol/geom';
import { isPoint } from '@l-ui/components/map/options/geometry';

const mapPointRef = ref()
const points1 = [
  { lng: 116.41, lat: 39.9, pointId: 21, pointName: '点1' },
  { lng: 116.57, lat: 39.92, pointId: 22, pointName: '点2' }
]
const points2 = [
  { lng: 116.41, lat: 39.6, pointId: 23, pointName: '点3' },
  { lng: 116.57, lat: 39.7, pointId: 24, pointName: '点4' }
]

onMounted(() => {
  mapPointRef.value.renderPoints(points1, {
    styleOpts: {
      type: 'circle'
    }
  })
  mapPointRef.value.renderPoints(points2, {
    styleOpts: {
      type: 'icon',
      iconSrc: () => 'mark'
    }
  })
})

const popoverData = ref<Record<string, any>>({});
const singleclick = (data: object, elementInfo: any) => {
  const popover = mapPointRef.value!.getPopover();
  if (data) {
    const geometry = elementInfo.geometry as Geometry;

    let coordinate: number[] = [];
    if (isPoint(geometry)) {
      const point = geometry as Point;
      coordinate = point.getCoordinates();
      popover.setPosition(coordinate);
      popoverData.value = data;
      return;
    }
  }

  popover.setPosition(undefined);
  popoverData.value = {};
}
</script>
<template>
  <l-map 
    ref="mapPointRef" 
    style="width: 100%; height: 300px"
    @singleclick="singleclick"
  >
    <template #popover>
      <div>点名称 : {{ popoverData.pointName }}</div>
      <div>点ID : {{ popoverData.pointId }}</div>
      <div>经度 : {{ popoverData.lng }}</div>
      <div>纬度 : {{ popoverData.lat }}</div>
    </template>
  </l-map>
</template>

<script setup lang="ts">
import { ref, onMounted } from 'vue'
const mapLineRef = ref()
const line = [
  { lng: 116.21, lat: 39.87, pointId: 21, pointName: '点1' },

  mapLineRef.value.renderLine(line, {
    startEndPoint: true,
    lineArrow: true,
    passPoint: true

绘制行政区 ​

显示代码
vue
<script setup lang="ts">
const mapAreaRef = ref()
const areas = [
  { code: '110000', value: 0.1 },
  { code: '130000', value: 0.5 },
  { code: '210000', value: 1 },
  { code: '650000', value: 0.8 },
  { code: '520000', value: 0.7 },
  { code: '310000', value: 0.6 },
  { code: '410000', value: 0.9 }
]

onMounted(() => {
  mapAreaRef.value.renderArea(areas, {
    adcode: '100000',
    url: '/geojson/100000_full.json',  // 本地geojson访问地址，或使用在线地址
    areaCallbackStyle: (item, info) => {
      if (item) {
        return {
          areaColor: `rgb(13,94,255,${item.value})`,
          lineColor: '#195e95',
          fontColor: '#fff'
        }
      }
    }
  })
})
</script>
<template>
  <l-map ref="mapAreaRef" :zoom="3" style="width: 100%; height: 300px"></l-map>
</template>

API ​

Attributes ​

参数说明类型可选值默认值
getLngKey从数据列表获取经度的 key,仅在绘制点、线、热力图有效string-lng
getLatKey从数据列表获取纬度的 key,仅在绘制点、线、热力图有效string-lat
getValeKey从数据列表获取权重的 key,仅在绘制热力图有效，数值范围 0-1string-value
getCodeKey从数据列表获取行政区划代码的 key,仅在绘制行政区有效string-code
zoom初始化地图缩放等级number-8
center初始化地图中心位置[number,number]-[116.41, 39.9]
pointClickZoom点击点后点位是否缩放boolean-false

Methods ​

参数说明类型参数
renderPoints绘制多个点function(points:[{"lng":116,"lat": 39}],pointsOptions
renderLine绘制一条线function(line:[{"lng":116,"lat": 39}],lineOptions:{styleOpts,lineArrow})
renderHeatmap绘制热力图function(points:[{"lng":116,"lat": 39}],heatmapOptions:{})
renderArea绘制行政区function(area:[{"code":110000}],areaOptions:{})

pointsOptions ​

参数说明类型默认值可选值
styleOpts-点样式object-
type点类型，图或者圆点stringiconicon/circle
circleRadius返回圆点大小，仅在 type=circle 有效function(point):number => 6-
circleColor返回圆点颜色，仅在 type=circle 有效function(point):string => #f00-
iconSrc返回图片点路径或者默认图，仅在 type=icon 有效function(point):string => markmark/
startMark/
passMark/
endMark/
图片路径
iconScale返回图片点缩放，仅在 type=icon 有效function(point):number => 1-
rotation返回图片点旋转角度，仅在 type=icon 有效function(point):number => 0-
anchor图片点基准点，仅在 type=icon 有效 ,从左上以 0 开始，0-1 之间的数[number,number][0.5,1]-

lineOptions ​

参数 说明类型默认值可选值

passPoint -是否绘制途径图标点booleanfalsefalse/true
styleOpts -路线样式object--
